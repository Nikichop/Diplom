Руководство пользования

Для того, чтобы структурировать исходные данные, вам потребуется:

1. Ввести в текстовые поля, в главном окне, данные и категории, по которым информация будет структурироваться.
2. Вписать ключ доступа к API в соответствующее поле.
3. Нажать кнопку отправить и ждать ответа.

В случае возникновения ошибок, система уведомит вас о том, в чем состоит их причина.

Касательно других возможностей приложения:
- В левом окне пользователь может выбрать: тему экрана, API большой языковой модели, ее версию, и формат вывода информации.
- В правом окне доступен выбор максимального количества, доступных для использования, токенов (базовой единицы текста, с которой работает LLM), кнопки загрузки и сохранения файлов,
а также кнопка руководства пользователя, на которую вы нажали ранее.
